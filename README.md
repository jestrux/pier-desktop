# Pier Desktop

## Getting Started

1. Install app dependencies

    ```sh
    yarn
    ```

2. Migrate database

    ```sh
    yarn migrate
    ```

3. Start the app in development mode

    ```sh
    yarn dev
    ```

4. Start the app in production mode

    ```sh
    yarn start
    ```

5. Package app. Uses [Electron Builder](https://www.electron.build/) to create a distributable package.

    ```sh
    yarn build
    ```
